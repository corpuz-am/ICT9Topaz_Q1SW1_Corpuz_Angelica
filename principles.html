Web Development Principles

Introduction Principles Video Resources

Web applications should work for everyone

  • Accessibility for everyone is prioritized.
  • Inclusive forms are made.
  • Applications are tested by real users.

Web applications should work everywhere

  • Web applications are made responsively.
  • The execution and capabilities is valued.
  • Develop offline-capable apps.
  • Make sure that the application is flexible.

Web applications should respect a user's privacy and security

  • Https is made use of everywhere.
  • The preference of a user is valued and respected.
  • Users are given complete information about how their data is being used.
  • Exporting data is authorized for users.
  • A user's data is secured and safe.

Web developers should be considerate of their peers

  • Observe if the code is documented.
  • Test your code.
  • Utilize source control and continuous integration.
  • When possible, try to consume and contribute to open source.
  • One should treat their other developers with respect and value.
  • Follow or administer ethical standards for open source projects.
